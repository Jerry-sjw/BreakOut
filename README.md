<h2>LearnOpenGL 2D game：BreakOut</h2><br>
Compile:<br>
1.Create a folder called build in the project<br>
  mkdir build<br>
2.Go to the build folder<br>
  cd build<br>
3.Compiling projects<br>
  cmake ..<br>
  cmake --build . --config Debug<br>
  
Run the game：<br>
  click BreakOut\build\bin\test.exe<br>
